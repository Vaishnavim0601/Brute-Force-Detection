# Brute Force Detection using Splunk

## Overview
Simulated RDP brute force attacks using Hydra in a controlled environment. Monitored and detected attack logs using Splunk, and generated alerts.

## Tools Used
- Hydra (for brute force simulation)
- Splunk (for log analysis and alerting)

## Outcome
- Detected and analyzed brute force attack patterns.
- Practiced incident detection and response.
- Improved understanding of SIEM monitoring and security operations.

## Skills Demonstrated
- Brute force attack simulation
- Log analysis
- SIEM (Splunk)
- Incident detection and response
