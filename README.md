# 🔑 Brute Force Attack Detection

## 📌 Overview

This project focuses on detecting brute force attacks by analyzing authentication logs and identifying repeated login attempts.

## 🎯 Objectives

* Simulate brute force attacks on RDP services
* Detect multiple failed login attempts
* Generate alerts for suspicious activity

## 🛠️ Tools & Technologies

* Hydra
* Splunk
* Kali Linux

## ⚙️ Methodology

1. Simulated RDP brute force attack using Hydra
2. Collected authentication logs
3. Imported logs into Splunk
4. Created queries to detect repeated login failures
5. Generated alerts for suspicious IP addresses

## 📊 Results

* Successfully detected brute force attempts
* Identified attacker IP and login patterns
* Generated real-time alerts in Splunk

## 🔐 Key Learnings

* Log analysis and SIEM monitoring
* Brute force attack detection techniques
* Alert creation in Splunk

## ⚠️ Disclaimer

This project was performed in a controlled lab environment for educational purposes only.
